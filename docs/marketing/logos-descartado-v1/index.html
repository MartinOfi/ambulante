8 opciones de logo · rosario · 2026

Cada logo es un archivo SVG editable. Podés abrirlos en Figma, Illustrator, Inkscape (gratis) para modificar colores, tipografía, proporciones. Las tipografías reales son Anton + Inter (Google Fonts). Si las instalás, los SVG se ven exactamente como diseñados.

Paleta usada en todos: Naranja Ambulante #F26B1F · Charcoal #1A1A1A · Cream #FAF6EE.

Mi recomendación

El #04 — Combo Pin con Carrito es la apuesta más fuerte: comunica el producto entero en 1 imagen (carrito + mapa), es memorable, y funciona en cualquier tamaño. Mantener el #07 — Monograma "A" como variante para favicon/app icon (las dos versiones del logo deben coexistir: el "completo" para branding grande y el "simple" para escalas chicas).

Apuestas alternativas si #04 no te convence:

→ #02 (Wordmark con A-pin) si querés algo más tipográfico y menos icónico.
→ #05 (Carrito Silueta) si priorizás conexión emocional con los vendedores ambulantes.
→ #01 (Wordmark Condensed) si querés máxima simplicidad y callejero puro.
