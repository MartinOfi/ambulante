> **Fase:** B3 — Boundaries de portabilidad (Repository + Facades + lint rule)
> **Goal de la fase:** Instalar la disciplina arquitectónica que permite swap de backend sin tocar features. Definir **dónde** vive Supabase y blindarlo con lint rules.
